# Digital Portfolio – Android Student Management App 📱

A mobile application that allows students to manage and visualize their academic information, including subjects, grades, feedback, and personal school data.

## 📘 Project Overview

This project was developed as part of the Mobile Development coursework. It allows users (students) to:
- Register and log in
- Add subjects, grades, and remarks
- Store and display data locally using SQLite
- Access a clean interface to view performance and feedback

## 🧠 Features

- CRUD operations on student data
- Dictionary of data entities defined
- UML class diagram & MLD included
- Data stored using a local SQLite database

## 🔧 Technologies

- Java (Android)
- SQLite
- XML (Android Layouts)
- UML 2.0
